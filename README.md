# RenewBerry Blog Platform

A modern, collaborative blog platform for creators and admins, built with React, Vite, TipTap, and Tailwind CSS.

## Features
- Rich text article editor with TipTap
- Creator and Admin dashboards
- Collaborative editing (Yjs, y-websocket)
- Category, tag, and author management
- Gallery and video support
- Responsive, accessible UI with Tailwind CSS
- Authentication and protected routes
- Analytics and earnings for creators

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Project Structure
- `src/` — Main source code (components, pages, contexts, hooks, etc.)
- `public/` — Static assets
- `index.html` — Main HTML entry
- `tailwind.config.js` — Tailwind CSS configuration
- `vite.config.ts` — Vite configuration
